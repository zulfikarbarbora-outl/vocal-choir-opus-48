# vocal-choir-opus-48

Real recorded crowd/choir singing single sustained vowels near F. Stereo XY-AKG214 @ 96 kHz. Sourced from Freesound pack 42098 (GameSoundCon2024 Walla, by Shane Vincent / ShangusBurger, CC0-1.0).

## About

This repo holds **48 kbps Opus previews** for fast streaming in the web DAW.
Opus at 48k delivers near-CD quality at ~35% the size of MP3 128k.

- **Library**: Vocal Choir (Real)
- **Files**: 15 Opus previews
- **Bitrate**: 48 kbps VBR
- **Container**: Ogg Opus (.opus)
- **License**: CC0-1.0
- **Source**: https://freesound.org/people/ShangusBurger/packs/42098/
- **Branch**: `main`
